The source code for this project is able to be compiled with JDK 17 from the terminal.

In order to run the program from the terminal, cd into the Assignment1 folder. Enter the commands:
(1) - javac ModuleOne.java
(2) - javac ModuleOneMethods.java
(3) - java ModuleOne (name) (binary number)
   * name - "Johnnie"
   * binary number - "10110111"
   ** For name and binary number command line variables, these are values I used. However, this program compiles with other variables values.
